- PostgreSQL
- JPA/Hibernate

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- Java JDK 17
- PostgreSQL 15
- Maven

### Installation

1. Clone the repository
```bash
git clone [repository-url]
```

2. Frontend Setup
```bash
cd frontend
npm install
npm start
```

3. Backend Setup
```bash
cd backend
mvn clean install
mvn spring-boot:run
```

4. Database Setup
- Create a PostgreSQL database named 'HellWeekCoffee'
- Update database credentials in `application.properties`
